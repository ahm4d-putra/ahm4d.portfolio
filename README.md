## Features

- **Modern UI/UX** — Clean design with Glassmorphism effects and dark mode support.
- **Fully Responsive** — Optimized for mobile, tablet, and desktop.
- **GSAP Animations** — Advanced scroll-triggered and entrance animations via GSAP ScrollTrigger.
- **Framer Motion** — Smooth, production-ready component transitions.
- **Multiple Languages** — EN/ID language switching powered by react-i18next.
- **Dynamic Sidebar** — Mobile-friendly animated navigation sidebar.
- **Interactive Playground** — Built-in Typing Speed Test with real-time WPM and accuracy tracking.
- **SEO Optimized** — Meta and Open Graph tags for better discoverability.
- **Custom SVG Icons** — Handcrafted icons to minimize external dependencies.

## Project Structure

```text
src/
├── components/       # Reusable UI components (Button, LanguageSwitcher, etc)
├── sections/         # Page sections (Hero, Navbar, Projects, Playground, etc)
├── locales/          # i18n translation files
│   ├── en/
│   │   └── translation.json
│   └── id/
│       └── translation.json
├── hooks/            # Custom React hooks
├── i18n.js           # i18next configuration
├── App.jsx           # Root component
├── main.jsx          # Entry point
└── index.css         # Global styles & Tailwind directives
public/
├── favicon.svg
└── og-image.png
```
